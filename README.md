# Learning C

Topics covered
- Printing to Console

## Print to Console
```c
printf("Hello World!\n");

int myInt = 1;
float myFloat = 1.0;
double myDouble = 1.5;
char myChar = 'a';
char myString[] = "abc";

printf("%d, %f, %lf, %c, %s\n", myInt, myFloat, myDouble, myChar, myString);
printf("%i, %F\n", myInt, myFloat);
```
